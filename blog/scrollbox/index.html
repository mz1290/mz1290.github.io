Ingredients:

  • A blank HTML document
  • A blank CSS document
  • Some dummy text. Checkout out lipsum.com

Recipe

  1. Setup your HTML document with the basic HTML skeleton code
  2. Link your CSS style sheet to the HTML code
  3. Add a new div element within the HTML body and assign it an id=scrollbox (or whatever you want)
  4. Generate a large blob of text and paste it within the div
  5. In your CSS add a style block for #scrollbox (or whatever id you gave the element)
  6. The key CSS elements you need to define are as follows:
    • width
    • height
    • overflow: auto
    • border

And that's it. If you were able to keep up, you should be looking at a shiny new scrollbox. You're welcome.
